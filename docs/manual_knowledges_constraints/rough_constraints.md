1. 糖代谢约束 (Glucose Homeostasis)
肝脏是血糖的“调节器”，其核心逻辑是：空腹产糖，饭后储糖。
•	空腹血糖上限： 在正常生理状态（非糖尿病模型）下，空腹血糖应稳定在 3.9 – 6.1 mmol/L。
•	餐后血糖峰值： 即使在大负荷摄入糖分后，正常肝脏与胰岛素协同作用，餐后 2 小时血糖不应超过 7.8 mmol/L。
•	糖异生/糖原分解逻辑： 
o	当血糖 > 5.5 mmol/L 时，肝脏净输出葡萄糖应趋近于 0（停止糖异生）。
o	当血糖 < 4.0 mmol/L 时，肝脏必须启动糖原分解或糖异生。
•	糖原储备上限： 肝糖原约占肝脏重量的 5%–8%（成人约 70g–100g），超过此阈值，多余的糖必须转化为脂肪（VLDL）。

2. 氮代谢与尿素循环 (Nitrogen/Urea Cycle)
•	血氨浓度限制： 正常血氨应保持在极低水平，通常 < 60 μmol/L（或者 10–35 μmol/L）。如果模型中氨浓度持续上升，说明肝脏清除氨的能力不足。
•	尿素产出逻辑： 肝脏是人体唯一能大规模合成尿素的器官。输入端的氨（NH3）与输出端的尿素（Urea）必须满足质量守恒。
•	血尿素氮： 正常范围在 2.5 – 7.1 mmol/L。

3. 胆红素代谢 (Bilirubin Metabolism)
•	总胆红素（TBIL）上限： 正常值应 < 21 μmol/L (或 1.2 mg/dL)。
•	如果模型显示“间接胆红素”异常升高，通常模拟的是肝细胞受损、处理能力达到瓶颈或者发生溶血。
•	直接胆红素（DBIL）： 应 < 7 μmol/L。
________________________________________
4. 合成功能与蛋白质 (Synthetic Function)
•	白蛋白（Albumin）： 肝脏每天合成约 12g – 15g 白蛋白。血浆浓度应维持在 35 – 55 g/L。若白蛋白合成率为 0 或负数，判定为肝衰竭模式。
•	凝血因子： 除因子 VIII 外，几乎所有凝血因子都在肝脏合成。虽然数值难以直接作为测试，但可以通过 凝血酶原时间 (PT) 的稳定性来间接约束，PT 正常值通常在11 – 14 秒，缺点是难以模拟。

5. 脂代谢约束 (Lipid Metabolism)
•	胆固醇总量： 肝脏合成人体 70%–80% 的内源性胆固醇。总胆固醇（TC）一般应 < 5.2 mmol/L。肝衰竭 / 晚期硬化< 2.5 - 3.0 mmol/L (低值)。
•	甘油三酯输出： 肝脏不长期储存脂肪（正常肝细胞脂肪含量 < 5%）。如果模型模拟中脂肪酸输入远超 VLDL 输出，应触发“脂肪肝”逻辑警告。
