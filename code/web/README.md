# 肝脏代谢数字化孪生系统 - 网页使用说明

本系统是一个交互式的肝脏代谢动态模拟平台，支持多组并发模拟对比、实时数据可视化及深度追溯分析。

## 1. 核心功能布局

网页分为四个主要区域：
- **左侧栏 (模拟配置)**：控制全局模拟参数、事件调度、重放进度及系统显示设置。
- **中上部 (模拟组展示区)**：显示各模拟对比组的仪表盘和分析结果。
- **中下部 (反应网络区)**：以层次化结构展示肝脏代谢模块及其内部反应。
- **右侧栏 (物质浓度面板)**：实时监控代谢物浓度，并支持搜索与预警。

---

## 2. 快速入门指南

### 2.1 创建与配置模拟组
1. 点击左侧栏的 **“+ 新建模拟对比组”** 可以创建多个平行的模拟。
2. **重命名**：直接点击模拟组顶部的名称输入框即可修改（如“正常组”、“肝损伤组”）。
3. **环境参数**：点击组内的 **“⚙ 环境参数”** 按钮，可为该组设置独立的生理参数（如肝功能系数、胰岛素降解活性）。

### 2.2 调度事件
在左侧栏 **“事件调度”** 区域，点击 **“+ 添加注入事件”**，设置注入时间、物质名称及剂量。这些事件将同步作用于所有模拟组。

### 2.3 启动模拟
点击 **“▶ 开始模拟”**。系统将实时通过 WebSocket 接收各组模拟数据，并同步更新所有可视化图表。

---

## 3. 交互与分析

### 3.1 仪表盘操作
- **实时监测**：仪表盘左侧显示代谢物与激素曲线，右侧显示选定的反应速率。
- **时间分析**：
    1. 单击仪表盘内的物质浓度图，可标记时间点 **t1**。
    2. 再次单击可标记 **t2**，划定一个 **分析时间段**。
    3. 系统将自动计算该时段内各反应对该物质浓度变化的 **累计贡献量**，并在下方的“追溯分析”区域显示。
- **分析区域折叠**：点击“追溯分析”标题可折叠/展开详细数据，方便对比。

### 3.2 反应网络与物质面板
- **组切换**：点击某个模拟组的任意位置，该组将变为“选中状态”（蓝色边框）。底部的反应框和右侧的物质数值将**即时切换**为该组的数据。
- **动态增删图表**：
    - **双击** 底部反应卡片：在所有组的仪表盘中添加/移除该反应速率曲线。
    - **双击** 右侧物质卡片：在所有组的仪表盘中添加/移除该物质浓度曲线。
- **追溯连线**：**单击** 右侧物质卡片，系统将显示连接至该物质的所有反应通量线（绿色表示消耗，红色表示产出），线条粗细代表即时速率。

### 3.3 重放与查看
- 使用左侧栏的 **“模拟重放进度”** 滑块可以回溯已完成的模拟过程。
- 仪表盘上的分析标记和铺色区域会随重放进度动态调整显示。

---

## 4. 个性化设置
- **网页底色主题**：在左侧栏开启/关闭 **“明亮模式”** 开关，可实时切换黑白主题。
- **物质预警**：当物质浓度超出正常生理范围（定义于 `metabolite_constraints.yaml`）时，右侧卡片将出现红色呼吸灯特效预警。
- **搜索功能**：在右侧栏搜索框输入名称，可快速发现并监控库中数以百计的代谢物。

---

## 5. 技术依赖
- **后端**：Flask, Flask-SocketIO, PyYAML, Eventlet.
- **前端**：Plotly.js, Socket.IO Client.
